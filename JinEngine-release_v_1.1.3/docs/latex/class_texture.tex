\doxysubsection{Detailed Description}
Encapsulates an Open\+GL texture object and its associated metadata. 

\doxylink{class_texture}{Texture} manages the lifetime of an Open\+GL texture ID along with its width, height, and channel count. It provides interfaces for binding the texture for sampling or image load/store operations.

\doxylink{class_texture}{Texture} creation can be driven from a file path, raw pixel data, or an externally supplied Open\+GL texture ID. Low-\/level update and generation functions are restricted to friend classes such as \doxylink{class_material}{Material} and \doxylink{class_render_manager}{Render\+Manager}.
