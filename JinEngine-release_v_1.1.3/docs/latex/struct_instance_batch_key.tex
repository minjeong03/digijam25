\doxysubsection{Detailed Description}
Key structure used for grouping Game\+Objects into instanced rendering batches. 

\doxylink{struct_instance_batch_key}{Instance\+Batch\+Key} represents a unique combination of rendering-\/relevant resources that must match in order for multiple Game\+Objects to be rendered together using instanced rendering.

The key consists of\+:
\begin{DoxyItemize}
\item \doxylink{class_mesh}{Mesh} pointer (geometry)
\item \doxylink{class_material}{Material} pointer (shader + texture state)
\item \doxylink{class_sprite_sheet}{Sprite\+Sheet} pointer (optional sprite animation source)
\end{DoxyItemize}

Two Game\+Objects can only be placed in the same instanced batch if all three pointers are identical.

This structure is designed to be used as a key in associative containers such as std\+::unordered\+\_\+map and std\+::map.
