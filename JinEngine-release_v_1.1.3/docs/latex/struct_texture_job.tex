\doxysubsection{Detailed Description}
Load request describing a texture to be loaded asynchronously. 

\doxylink{struct_texture_job}{Texture\+Job} contains all parameters needed by the worker thread to load raw pixel data from disk and prepare a \doxylink{struct_texture_result}{Texture\+Result}.

The tag is used by the main thread to register the loaded texture into the engine\textquotesingle{}s resource registry.
