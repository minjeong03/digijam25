\doxysubsection{Detailed Description}
Describes configuration parameters used when creating or updating a texture. 

\doxylink{struct_texture_settings}{Texture\+Settings} encapsulates common Open\+GL texture state such as filtering, wrapping, and mipmap generation. It is passed to texture creation and update routines to ensure consistent configuration.
