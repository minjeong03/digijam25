\doxysubsection{Detailed Description}
Broad-\/phase collision structure using spatial hashing. 

\doxylink{class_spatial_hash_grid}{Spatial\+Hash\+Grid} divides world space into square cells (cell\+Size). Objects are inserted into one or more cells based on their positions.

During Compute\+Collisions\+:
\begin{DoxyItemize}
\item The grid enumerates objects per cell and generates candidate pairs.
\item The on\+Collision callback is invoked for each candidate pair (typically leading to narrow-\/phase collider checks).
\end{DoxyItemize}

Large objects that exceed cell-\/based assumptions may be tracked separately in large\+Objects, allowing the system to handle them without excessive multi-\/cell insertion.
