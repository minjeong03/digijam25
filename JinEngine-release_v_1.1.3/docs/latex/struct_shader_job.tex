\doxysubsection{Detailed Description}
Load request describing a shader program to be loaded asynchronously. 

\doxylink{struct_shader_job}{Shader\+Job} groups multiple stage files under a single tag. The worker thread reads each file into memory and produces \doxylink{struct_shader_result}{Shader\+Result} containing source strings per stage.
