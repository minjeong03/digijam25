\doxysubsection{Detailed Description}
Base class for all in-\/game objects with gameplay behavior. 

\doxylink{class_game_object}{Game\+Object} is a concrete specialization of \doxylink{class_object}{Object} that represents entities participating in gameplay logic.

It is constructed with \doxylink{_object_8h_a842c5e2e69277690b064bf363c017980a4504e1ed59cd9732b8a844e5424e6f13}{Object\+Type\+::\+GAME} and is intended to be managed by \doxylink{class_object_manager}{Object\+Manager} within a \doxylink{class_game_state}{Game\+State}.

All lifecycle functions are provided as overridable hooks and are empty by default, allowing derived classes to implement only the behavior they require.
