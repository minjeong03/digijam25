\doxysubsection{Detailed Description}
Cached glyph metrics and atlas UV coordinates for a single Unicode codepoint. 

This struct stores the data needed to layout and render a glyph quad\+:
\begin{DoxyItemize}
\item size\+: bitmap width/height in pixels
\item bearing\+: left/top bearing in pixels from baseline
\item advance\+: Free\+Type advance value in 26.\+6 fixed-\/point format (advance \texorpdfstring{$>$}{>}\texorpdfstring{$>$}{>} 6 = pixels)
\item uv\+Top\+Left / uv\+Bottom\+Right\+: normalized UV region in the font atlas texture
\end{DoxyItemize}

\doxylink{struct_glyph}{Glyph} instances are stored in \doxylink{class_font_afbb7e0139ebb5cfbdd3c51a9bd070484}{Font\+::glyphs} and are populated during baking via \doxylink{class_font_aa3f349ce0e57bfabb0ef4cea1431a1d6}{Font\+::\+Try\+Bake\+Glyph}.
