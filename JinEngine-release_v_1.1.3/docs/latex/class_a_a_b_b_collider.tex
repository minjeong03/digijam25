\doxysubsection{Detailed Description}
Axis-\/aligned bounding box (AABB) collider implementation. 

\doxylink{class_a_a_b_b_collider}{AABBCollider} stores both base and scaled half-\/extents. When transform scale syncing is enabled, scaled\+Half\+Size is updated relative to base\+Half\+Size in Sync\+With\+Transform\+Scale.

The constructor receives full size and stores half size internally.
