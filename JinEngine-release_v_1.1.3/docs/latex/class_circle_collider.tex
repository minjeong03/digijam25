\doxysubsection{Detailed Description}
Circle collider implementation. 

\doxylink{class_circle_collider}{Circle\+Collider} stores both a base radius and a scaled radius. When transform scale syncing is enabled, scaled\+Radius is updated relative to base\+Radius in Sync\+With\+Transform\+Scale.

The constructor receives a size and converts it to radius by dividing by 2.
