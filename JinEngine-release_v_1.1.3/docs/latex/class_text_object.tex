\doxysubsection{Detailed Description}
Scene object responsible for rendering text within the standard \doxylink{class_object}{Object} pipeline. 

\doxylink{class_text_object}{Text\+Object} is a specialized \doxylink{class_object}{Object} implementation for in-\/world text rendering. Unlike HUD or immediate-\/mode text systems, \doxylink{class_text_object}{Text\+Object} participates in the normal \doxylink{class_object}{Object} lifecycle (Init / Update / Draw / Free) and render ordering.

Internally, \doxylink{class_text_object}{Text\+Object} owns a dynamically generated \doxylink{class_mesh}{Mesh} representing the glyph geometry of the current text. The mesh is regenerated whenever the text content changes or when the associated \doxylink{class_font}{Font}\textquotesingle{}s atlas version changes.

\doxylink{class_text_object}{Text\+Object} deliberately disables access to generic material, mesh, and animation interfaces inherited from \doxylink{class_object}{Object} to prevent misuse. Text rendering is tightly coupled to Font-\/driven mesh generation and a dedicated text material/shader setup.
