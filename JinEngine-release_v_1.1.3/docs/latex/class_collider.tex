\doxysubsection{Detailed Description}
Abstract base collider that supports broad-\/phase + narrow-\/phase collision checks. 

\doxylink{class_collider}{Collider} represents a shape attached to an \doxylink{class_object}{Object} and provides the core interfaces used by \doxylink{class_object_manager}{Object\+Manager} and \doxylink{class_spatial_hash_grid}{Spatial\+Hash\+Grid}\+:
\begin{DoxyItemize}
\item point containment test (Check\+Point\+Collision)
\item shape-\/vs-\/shape collision test (Check\+Collision + Dispatch\+Against)
\item broad-\/phase approximation (Get\+Bounding\+Radius)
\item optional transform scale synchronization (Sync\+With\+Transform\+Scale)
\item debug visualization (Draw\+Debug)
\end{DoxyItemize}

Narrow-\/phase collision is implemented using double dispatch\+: Check\+Collision(other) calls other-\/\texorpdfstring{$>$}{>}Dispatch\+Against(\texorpdfstring{$\ast$}{*}this\+Concrete) or an equivalent path, allowing Circle-\/vs-\/\+AABB and AABB-\/vs-\/\+Circle to be resolved without RTTI/dynamic\+\_\+cast.

Offset is stored separately from the owner\textquotesingle{}s transform and is typically used to shift the collider relative to the owner\textquotesingle{}s origin.

This class is not default-\/constructible; an owner must be provided.
