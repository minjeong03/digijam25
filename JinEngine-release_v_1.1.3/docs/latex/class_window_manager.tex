\doxysubsection{Detailed Description}
Manages the application\textquotesingle{}s primary GLFW window and related OS-\/facing behaviors. 

\doxylink{class_window_manager}{Window\+Manager} owns the GLFWwindow handle and tracks the current window size and presentation state (windowed vs fullscreen). It also stores a background clear color used by the engine\textquotesingle{}s per-\/frame clear routine.

Resize handling is driven by the GLFW framebuffer size callback (framebuffer\+\_\+size\+\_\+callback), which updates this manager\textquotesingle{}s cached width/height, updates all cameras\textquotesingle{} screen sizes through the active \doxylink{class_game_state}{Game\+State}\textquotesingle{}s \doxylink{class_camera_manager}{Camera\+Manager}, notifies the \doxylink{class_render_manager}{Render\+Manager} about the resize, and finally invokes any user-\/registered resize callbacks via \doxylink{class_window_manager_ac494c42b664babdcd3871974eae6fb52}{Notify\+Resize()}.

Most lifecycle and per-\/frame operations (Init/\+Swap\+Buffers/\+Clear\+Screen/\+Poll\+Events/\+Free) are private and are intended to be orchestrated by \doxylink{class_jin_engine}{Jin\+Engine}.

\begin{DoxyItemize}

\item Initializes GLAD for Open\+GL function loading.
\item Enables Open\+GL debug output and installs a debug message callback (non-\/notification severities).
\item Sets the GLFW user pointer to the provided \doxylink{class_jin_engine}{Jin\+Engine} instance.

Toggles fullscreen mode using the primary monitor. 

When enabling fullscreen, the current windowed position and size are saved and the window is switched to the primary monitor\textquotesingle{}s current video mode. When disabling, the saved windowed position/size are restored.

If the window is null or the requested state matches the current state, this function does nothing.
