\doxysubsection{Detailed Description}
Base interface for all runtime entities managed by \doxylink{class_object_manager}{Object\+Manager} and rendered by \doxylink{class_render_manager}{Render\+Manager}. 

\doxylink{class_object}{Object} defines the engine-\/level lifecycle and the common component set used by the renderer and collision system\+:
\begin{DoxyItemize}
\item Lifecycle\+: Init -\/\texorpdfstring{$>$}{>} Late\+Init -\/\texorpdfstring{$>$}{>} Update -\/\texorpdfstring{$>$}{>} Draw -\/\texorpdfstring{$>$}{>} Free -\/\texorpdfstring{$>$}{>} Late\+Free
\item Rendering\+: \doxylink{class_transform2_d}{Transform2D}, Material\texorpdfstring{$\ast$}{*}, Mesh\texorpdfstring{$\ast$}{*}, render layer tag, per-\/object color, optional \doxylink{class_sprite_animator}{Sprite\+Animator}
\item Collision\+: optional \doxylink{class_collider}{Collider} and collision category/mask filtering
\end{DoxyItemize}

Ownership notes\+:
\begin{DoxyItemize}
\item \doxylink{class_object_manager}{Object\+Manager} owns \doxylink{class_object}{Object} instances via std\+::unique\+\_\+ptr$<$\+Object$>$.
\item \doxylink{class_material}{Material} and \doxylink{class_mesh}{Mesh} are referenced as raw pointers and are owned/managed by \doxylink{class_render_manager}{Render\+Manager}.
\item \doxylink{class_sprite_animator}{Sprite\+Animator} and \doxylink{class_collider}{Collider} are owned by the \doxylink{class_object}{Object} via std\+::unique\+\_\+ptr.
\end{DoxyItemize}

Camera-\/space behavior\+:
\begin{DoxyItemize}
\item Objects can be configured to ignore the active camera and render in screen space.
\item When ignoring the camera, a reference\+Camera may be used to compute world transforms consistently.
\end{DoxyItemize}

Visibility and lifetime\+:
\begin{DoxyItemize}
\item is\+Alive controls whether the object is considered active; killed objects are freed and removed by \doxylink{class_object_manager}{Object\+Manager}.
\item is\+Visible controls whether the object participates in rendering and (typically) debug visualization. 
\end{DoxyItemize}
