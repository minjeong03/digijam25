\doxysubsection{Detailed Description}
Free\+Type-\/backed font that generates text meshes using a dynamically baked atlas. 

\doxylink{class_font}{Font} loads a TTF face using Free\+Type and maintains a single-\/channel (GL\+\_\+\+RED) atlas texture that is populated on demand.

Key behaviors based on the implementation\+:
\begin{DoxyItemize}
\item The input string is treated as UTF-\/8 and decoded into Unicode codepoints.
\item \doxylink{class_font_aa3f349ce0e57bfabb0ef4cea1431a1d6}{Font\+::\+Try\+Bake\+Glyph} lazily renders a glyph via FT\+\_\+\+Load\+\_\+\+Char(..., FT\+\_\+\+LOAD\+\_\+\+RENDER) and uploads it into the atlas using gl\+Texture\+Sub\+Image2D.
\item If the atlas becomes full, \doxylink{class_font_a225429b93aa4fe0d76b2fecd9ac88142}{Font\+::\+Expand\+Atlas} doubles the atlas size and re-\/bakes all previously baked glyphs into the new texture, then increments atlas\+Version.
\item \doxylink{class_font_a3a3963ea1c039525cbf922f52ae03670}{Font\+::\+Generate\+Text\+Mesh} builds a quad per glyph (4 vertices, 6 indices) and supports horizontal/vertical alignment for multi-\/line text.
\end{DoxyItemize}

The \doxylink{class_font}{Font} also owns a \doxylink{class_material}{Material} that references the atlas texture via the uniform name "{}u\+\_\+\+Font\+Texture"{} and sets a default "{}u\+\_\+\+Color"{} uniform.

Generates a mesh containing quads for the given UTF-\/8 text. 

This function\+:
\begin{DoxyItemize}
\item Splits the input by \textquotesingle{}~\newline
\textquotesingle{} into lines.
\item Pre-\/bakes glyphs for each line to compute accurate line widths.
\item Applies horizontal alignment per line by shifting the X cursor\+:
\begin{DoxyItemize}
\item Left\+: no shift
\item Center\+: -\/line\+Width \texorpdfstring{$\ast$}{*} 0.\+5
\item Right\+: -\/line\+Width
\end{DoxyItemize}
\item Applies vertical alignment by shifting the starting Y cursor based on total height.
\item For each codepoint, appends a quad (4 vertices, 6 indices) using the glyph\textquotesingle{}s bearing, size, and atlas UVs.
\end{DoxyItemize}

The returned \doxylink{class_mesh}{Mesh} is allocated with new and must be managed/freed by the caller according to the engine\textquotesingle{}s ownership conventions.
