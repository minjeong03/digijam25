\doxysubsection{Detailed Description}
2D camera providing view transformation and visibility tests. 

\doxylink{class_camera2_d}{Camera2D} represents a simple orthographic-\/style camera for 2D rendering. It maintains position, zoom, and screen size information and provides\+:
\begin{DoxyItemize}
\item View matrix generation for rendering
\item Screen size management
\item World-\/space visibility testing for basic view culling
\end{DoxyItemize}

The camera operates in world space using a position and a uniform zoom factor. Screen size is used to determine the visible region in world coordinates.
