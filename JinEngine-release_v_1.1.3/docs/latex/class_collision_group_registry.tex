\doxysubsection{Detailed Description}
Registry that assigns bit indices to collision group tags. 

\doxylink{class_collision_group_registry}{Collision\+Group\+Registry} maintains a stable mapping between string tags (e.\+g., "{}\+Player"{}, "{}\+Enemy"{}, "{}\+Pickup"{}) and a unique bit index.

The registry supports\+:
\begin{DoxyItemize}
\item Allocating a new bit when a previously unseen tag is requested
\item Reverse lookup from bit index back to tag for debugging/inspection
\end{DoxyItemize}

This is typically used to build collision filtering masks (group membership and interaction rules).
