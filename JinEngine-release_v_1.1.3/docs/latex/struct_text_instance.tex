\doxysubsection{Detailed Description}
Lightweight data container representing a text rendering request. 

\doxylink{struct_text_instance}{Text\+Instance} holds a reference to a \doxylink{class_font}{Font} and the UTF-\/8 text string to be rendered. It does not own the \doxylink{class_font}{Font} resource and is intended to be used as a mutable payload that can trigger mesh regeneration when its contents change.
