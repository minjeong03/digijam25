\doxysubsection{Detailed Description}
Manages multiple \doxylink{class_camera2_d}{Camera2D} instances identified by string tags. 

\doxylink{class_camera_manager}{Camera\+Manager} owns \doxylink{class_camera2_d}{Camera2D} objects via std\+::unique\+\_\+ptr and provides\+:
\begin{DoxyItemize}
\item Registration of cameras by tag
\item Lookup by tag
\item Active camera selection by tag
\item Bulk screen size propagation (useful on window resize)
\end{DoxyItemize}

The active camera is tracked by its tag (active\+Camera\+Tag). If the active tag does not exist in the map, Get\+Active\+Camera returns nullptr.
