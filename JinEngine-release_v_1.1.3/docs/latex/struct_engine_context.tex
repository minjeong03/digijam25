\doxysubsection{Detailed Description}
Aggregates pointers to core engine subsystems. 

\doxylink{struct_engine_context}{Engine\+Context} acts as a lightweight dependency container that is passed throughout the engine during update, render, and lifecycle calls.

It provides unified access to major engine systems such as\+:
\begin{DoxyItemize}
\item State management
\item Window and input handling
\item Rendering
\item Audio
\item Core engine control
\end{DoxyItemize}

\doxylink{struct_engine_context}{Engine\+Context} does not own these systems. It only stores raw pointers that are managed and lifetime-\/controlled by \doxylink{class_jin_engine}{Jin\+Engine}.

This structure is intentionally simple and copyable, and is commonly passed by const reference to avoid tight coupling between systems.
