Map structure used for organizing instanced rendering batches per render layer. 

The outer map key (int) typically represents a render layer or render order.

Each layer contains an unordered\+\_\+map where\+:
\begin{DoxyItemize}
\item The key is \doxylink{struct_instance_batch_key}{Instance\+Batch\+Key} (shared render state)
\item The value is a list of \doxylink{class_game_object}{Game\+Object} pointers that can be rendered together using instanced rendering.
\end{DoxyItemize}

This structure allows the renderer to\+:
\begin{DoxyItemize}
\item Group objects by layer
\item Further batch objects by identical render resources
\item Minimize draw calls during rendering 
\end{DoxyItemize}
