Variant type used to store strongly-\/typed uniform values for a \doxylink{class_material}{Material}. 

\doxylink{class_material}{Material} stores per-\/uniform data in a map of (uniform name -\/\texorpdfstring{$>$}{>} \doxylink{_material_8h_a06229a425f27fcb99ce02ae045b56e0d}{Uniform\+Value}). When binding, \doxylink{class_material}{Material} serializes these values and forwards them to Shader\+::\+Send\+Uniform(...) using the correct overload based on the active variant type.

Supported types are limited to the most common scalar/vector/matrix uniform types used by the engine.
