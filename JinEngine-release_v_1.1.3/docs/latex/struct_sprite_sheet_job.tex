\doxysubsection{Detailed Description}
Load request describing a sprite sheet resource to be created/registered. 

\doxylink{struct_sprite_sheet_job}{Sprite\+Sheet\+Job} references an existing texture by tag and specifies the sprite sheet grid division (frameW, frameH).

Note\+: This job type exists in \doxylink{_async_resource_loader_8h_ae1b8cce382c5ee957e16be16b2cded39}{Load\+Request}, but there is no corresponding Sprite\+Sheet\+Result in \doxylink{_async_resource_loader_8h_a6a60ad85728d8591ed1f4246e5d3af21}{Load\+Result} in this header. It is expected that the worker/main-\/thread integration handles sprite sheet registration separately.
