\doxysubsection{Detailed Description}
Top-\/level engine facade that owns core managers and drives the main game loop. 

\doxylink{class_jin_engine}{Jin\+Engine} is the central orchestrator that\+:
\begin{DoxyItemize}
\item Owns and initializes the window, input, render, sound, and state systems.
\item Builds an \doxylink{struct_engine_context}{Engine\+Context} that exposes pointers/references to those systems for use by Game\+State/\+Object code.
\item Runs the main loop (Run) until \doxylink{class_jin_engine_a0fff451b96b00ef74a4a9975239adfdd}{Request\+Quit()} is called.
\end{DoxyItemize}

Lifetime\+:
\begin{DoxyItemize}
\item \doxylink{class_jin_engine_af2b70669383ccdbbc05309c7e4eb50db}{Init()} creates/initializes core subsystems and sets up \doxylink{struct_engine_context}{Engine\+Context}.
\item \doxylink{class_jin_engine_ad8355b55406f0be0b2f39b910ff7913e}{Run()} repeatedly updates input, state, rendering, and window events until should\+Run becomes false.
\item \doxylink{class_jin_engine_a00fa0799a110bb78192a6a643f55d309}{Free()} performs shutdown/cleanup in the correct order.
\end{DoxyItemize}

Debug draw\+:
\begin{DoxyItemize}
\item \doxylink{class_jin_engine_abb48d74bdf8780af4dc942c60752d6ae}{Render\+Debug\+Draws()} toggles engine-\/wide debug rendering.
\item \doxylink{class_jin_engine_a35998b207827eaccd30459251853c9da}{Should\+Render\+Debug\+Draws()} is mainly used internally by rendering/debug systems to branch behavior. 
\end{DoxyItemize}

Initializes all core subsystems and prepares the engine to run. 

Expected responsibilities\+:
\begin{DoxyItemize}
\item Initialize \doxylink{class_window_manager}{Window\+Manager} with the requested window size.
\item Initialize \doxylink{class_render_manager}{Render\+Manager} and create required default resources (fallback shader/texture/mesh/material).
\item Initialize \doxylink{class_input_manager}{Input\+Manager} with a valid window handle.
\item Initialize \doxylink{class_sound_manager}{Sound\+Manager} audio backend.
\item Initialize \doxylink{class_state_manager}{State\+Manager} to a valid initial state (or a loading/boot state), depending on engine setup.
\item Build \doxylink{struct_engine_context}{Engine\+Context} so that it exposes the above systems consistently to Game\+State/\+Object code.
\end{DoxyItemize}

If any subsystem fails to initialize, \doxylink{class_jin_engine_af2b70669383ccdbbc05309c7e4eb50db}{Init()} returns false and the engine should not \doxylink{class_jin_engine_ad8355b55406f0be0b2f39b910ff7913e}{Run()}.
