Detailed Description

Axis-aligned bounding box (AABB) collider implementation.

AABBCollider stores both base and scaled half-extents. When transform scale syncing is enabled, scaledHalfSize is updated relative to baseHalfSize in SyncWithTransformScale.

The constructor receives full size and stores half size internally.
