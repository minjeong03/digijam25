Detailed Description

Key structure used for grouping GameObjects into instanced rendering batches.

InstanceBatchKey represents a unique combination of rendering-relevant resources that must match in order for multiple GameObjects to be rendered together using instanced rendering.

The key consists of:

  • Mesh pointer (geometry)
  • Material pointer (shader + texture state)
  • SpriteSheet pointer (optional sprite animation source)

Two GameObjects can only be placed in the same instanced batch if all three pointers are identical.

This structure is designed to be used as a key in associative containers such as std::unordered_map and std::map.
