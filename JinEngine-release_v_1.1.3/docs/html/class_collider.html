Detailed Description

Abstract base collider that supports broad-phase + narrow-phase collision checks.

Collider represents a shape attached to an Object and provides the core interfaces used by ObjectManager and SpatialHashGrid:

  • point containment test (CheckPointCollision)
  • shape-vs-shape collision test (CheckCollision + DispatchAgainst)
  • broad-phase approximation (GetBoundingRadius)
  • optional transform scale synchronization (SyncWithTransformScale)
  • debug visualization (DrawDebug)

Narrow-phase collision is implemented using double dispatch: CheckCollision(other) calls other->DispatchAgainst(*thisConcrete) or an equivalent path, allowing Circle-vs-AABB and AABB-vs-Circle to be resolved without RTTI/dynamic_cast.

Offset is stored separately from the owner's transform and is typically used to shift the collider relative to the owner's origin.

This class is not default-constructible; an owner must be provided.
