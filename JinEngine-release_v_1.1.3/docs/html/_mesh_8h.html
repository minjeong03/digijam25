enum class PrimitiveType
strong

Defines the primitive topology used when issuing draw calls for a Mesh.

PrimitiveType selects how the vertex/index data is interpreted by the GPU draw call. RenderManager/Mesh map this enum to the corresponding OpenGL primitive (e.g., Triangles -> GL_TRIANGLES).
