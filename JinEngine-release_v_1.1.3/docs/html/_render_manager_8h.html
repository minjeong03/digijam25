◆ RenderMap

using RenderMap = std::array<std::map<int, ShaderMap>, RenderLayerManager::MAX_LAYERS>

Layered render map used to batch and sort objects for rendering.

RenderMap is an array with RenderLayerManager::MAX_LAYERS entries. Each layer contains a depth-sorted map:

  • std::map<int, ShaderMap> where the int key is a quantized depth bin derived from Transform2D::GetDepth().

The render pipeline order is: 1) layer (0..MAX_LAYERS-1) 2) depth bin (ascending int) 3) shader 4) InstanceBatchKey (mesh/material/spritesheet grouping) 5) objects in the batch

◆ ShaderMap

using ShaderMap = std::unordered_map<Shader*, std::map<InstanceBatchKey, std::vector<Object*>>>

Maps a Shader pointer to depth-sorted instance batches.

The outer key is the Shader used for rendering. The value is a depth-ordered map (std::map) whose key is InstanceBatchKey, and whose value is a list of Object*.

This structure is produced by BuildRenderMap() and consumed by FlushDrawCommands().
