Detailed Description

Circle collider implementation.

CircleCollider stores both a base radius and a scaled radius. When transform scale syncing is enabled, scaledRadius is updated relative to baseRadius in SyncWithTransformScale.

The constructor receives a size and converts it to radius by dividing by 2.
