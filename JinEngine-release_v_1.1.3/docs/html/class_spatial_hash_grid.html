Detailed Description

Broad-phase collision structure using spatial hashing.

SpatialHashGrid divides world space into square cells (cellSize). Objects are inserted into one or more cells based on their positions.

During ComputeCollisions:

  • The grid enumerates objects per cell and generates candidate pairs.
  • The onCollision callback is invoked for each candidate pair (typically leading to narrow-phase collider checks).

Large objects that exceed cell-based assumptions may be tracked separately in largeObjects, allowing the system to handle them without excessive multi-cell insertion.
