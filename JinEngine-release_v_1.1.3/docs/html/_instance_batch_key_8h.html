using InstancedBatchMap = std::unordered_map<int, std::unordered_map<InstanceBatchKey, std::vector<GameObject*>>>

Map structure used for organizing instanced rendering batches per render layer.

The outer map key (int) typically represents a render layer or render order.

Each layer contains an unordered_map where:

  • The key is InstanceBatchKey (shared render state)
  • The value is a list of GameObject pointers that can be rendered together using instanced rendering.

This structure allows the renderer to:

  • Group objects by layer
  • Further batch objects by identical render resources
  • Minimize draw calls during rendering
