Variant type used to store strongly-typed uniform values for a Material.

Material stores per-uniform data in a map of (uniform name -> UniformValue). When binding, Material serializes these values and forwards them to Shader::SendUniform(...) using the correct overload based on the active variant type.

Supported types are limited to the most common scalar/vector/matrix uniform types used by the engine.
