Detailed Description

2D camera providing view transformation and visibility tests.

Camera2D represents a simple orthographic-style camera for 2D rendering. It maintains position, zoom, and screen size information and provides:

  • View matrix generation for rendering
  • Screen size management
  • World-space visibility testing for basic view culling

The camera operates in world space using a position and a uniform zoom factor. Screen size is used to determine the visible region in world coordinates.

nodiscard

Tests whether a circular object is inside the camera view.

This function performs a simple visibility test by checking whether a circle defined by a center position and radius intersects the camera's visible world-space rectangle.

It is commonly used for view frustum culling in 2D scenes.

Parameters
posCenter position of the object in world space.
radiusBounding radius of the object.
viewportSizeSize of the viewport in world units.
Returns
true if the object is considered visible; false otherwise.
